# Task Manager с LLM-функциями

Приложение для управления задачами с интеграцией LLM-функций, построенное на NextJS 16.

## 1. Краткое описание проекта и реализованных функций

### Управление задачами (CRUD)
- Создание, просмотр, редактирование и удаление задач
- Атрибуты: название, описание, приоритет, статус, категория, срок выполнения

### Фильтрация и поиск
- Фильтрация по статусу и приоритету
- Полнотекстовый поиск по названию и описанию
- Комбинируемые фильтры

### LLM-функции
- Умная категоризация задач
- Предложение приоритета
- Декомпозиция задач на подзадачи
- Генерация сводки по нагрузке

## 2. Пошаговые инструкции настройки среды

### Требования
- Node.js 18+

### Установка

```bash
npm install
```

### Переменные окружения

Создайте файл `.env.local` (опционально):

```env
GIGACHAT_CREDENTIALS=your_credentials
```

По умолчанию используются встроенные демо-credentials.

## 3. Инструкции запуска приложения

### Режим разработки

```bash
npm run dev
```

### Продакшн сборка

```bash
npm run build
npm start
```

### Адрес
- Приложение: http://localhost:3000

## 4. Архитектурные решения

### NextJS App Router
- Server Components для рендеринга
- API Routes для бэкенда
- Turbopack для быстрой сборки

### База данных
- SQLite через better-sqlite3
- Хранится в файле tasks.db

### UI
- Material UI (MUI)
- Адаптивный дизайн

## 5. Известные ограничения

1. Нет аутентификации
2. Нет пагинации
3. LLM требует интернет
